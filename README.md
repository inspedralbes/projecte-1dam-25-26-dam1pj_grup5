## Integrants:

## Nom del projecte:
Aplicació web de gestió d'incidències informàtiques

## Petita descripció:
Aquesta aplicació web em permet registrar incidències informàtiques d'un departament, assignar-les a un tècnic, registrar actuacions, consultar l'estat de la incidència i generar informes de seguiment.
